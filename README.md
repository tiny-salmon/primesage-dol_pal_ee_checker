# 프라임세이지 돌팔이체커

> PC 부품 시세를 자동으로 조회하고 성능 등급을 알려주는 지리는 신개념 앱
> 수리점 방문 전에 내 PC 하드웨어를 분석하고 돌팔이당하지 말기

---

## 실행 방법

`프라임세이지_돌팔이체커.exe` 더블클릭

- Python 설치 불필요
- 별도 설치 과정 없음

---

## 기능

| 탭 | 설명 |
|---|---|
| 내 PC 분석 | 내 컴퓨터 부품을 자동 감지하고 중고 시세와 성능 등급을 보여줌 |
| 최적 조립 추천 | 예산을 입력하면 그 돈으로 살 수 있는 최고 조합을 추천해줌 |
| 부품 비교 | 두 부품의 성능과 시세를 나란히 비교함 (기본값은 현재 부품) |
| 내 PC 보고서 | 분석 결과를 텍스트로 복사해서 굳이 써넣을필요 없이 복사가능 |

---

## 자주 물을꺼같은 질문

**Q. "Windows의 PC 보호" 경고가 뜬다?**  
"추가 정보" → "실행" 클릭하면 정상 실행됨

**Q. 백신이 차단한다**  
하드웨어를 읽는 과정에서 백신이 반응하여 이 프로그램을 차단할수있음, 백신 허용 목록에 추가하거나 잠시 끄고 실행해야함

**Q. 시세가 실제랑 다를수있다**     
중고 시세 특성상 ± 20% 차이가 날 수 있음 하지만 실제로 별 차이는 없으니 안심하고 써도됨
